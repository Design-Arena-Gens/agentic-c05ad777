## First Light in the Barn

This project delivers a polished, stage-ready script that dramatizes the birth of a calf while highlighting the practical steps of bovine labor. Built with Next.js, it presents characters, scene-by-scene beats, and production notes in a cinematic layout suitable for theatre, film, or educational video planning.

### Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the script experience.

### Project Structure

- `src/app/page.tsx` – Script content and structured presentation.
- `src/app/page.module.css` – Warm, atmospheric styling inspired by barn lighting.
- `src/app/layout.tsx` – Global metadata and font configuration.

### Deployment

Ready for instant deployment to Vercel (`vercel deploy --prod`), producing a static site that can be shared with cast and crew or embedded in production documents.
